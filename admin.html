Model thresholds (global — every device)
Heart & murmur (probability ≥)
↑ value = FEWER flags (stricter) · ↓ value = MORE flags (more sensitive)
Lung sounds (probability ≥)
↑ value = FEWER flags (stricter) · ↓ value = MORE flags (more sensitive)
Acoustic tests
Cough: ↓ delta = counts MORE coughs. Blow test/FET (flags obstruction if the breath lasts ≥ the value): ↑ = FEWER flags. Breath count/SBCT & Ahh time/MPT (flag if the result is BELOW the value): ↑ = flags MORE people.
Chest X-ray findings (probability ≥)
TB = tuberculosis pattern · Cardiomegaly = enlarged heart · Effusion = pleural fluid · Pneumonia = infection pattern · Consolidation = solid/filled lung · Nodule = lung nodule/mass · Pneumothorax = collapsed lung (urgent) · Fibrosis = lung scarring · Pleural thick. = thickened pleura
A flag = the app marks that finding as positive. Changes apply to every device (online & offline) on its next recording. Use the per-section notes above for which direction makes each test stricter.
Research validation mode
When ON, each VascAge scan in the app shows ECG: AF / Sinus / Other buttons that log the app's flag vs your ECG-confirmed truth to Supabase (af_validation table). Only you (admin) see this switch; frontline users never do.
